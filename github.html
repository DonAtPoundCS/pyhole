Submitting Code¶

Follow the procedure below when submitting code.

Standard GitHub Workflow¶

Start off my forking [1] pyhole to your account on GitHub. From here you can make your changes:

git clone git://github.com/<GitHub ID>/pyhole.git
cd pyhole

...make changes...

./tools/run_tests.sh
git commit -a -m "Detailed commit message."
git push

When done, open a Pull Request [2] on GitHub.

[1]http://help.github.com/fork-a-repo/
[2]http://help.github.com/send-pull-requests/
